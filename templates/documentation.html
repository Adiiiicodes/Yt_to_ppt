Project Documentation

This tool converts a YouTube video into a PowerPoint presentation by following these steps:

  1. The YouTube video is transcribed.
  2. The transcription is converted into a JSON format using the Groq API.
  3. The JSON is sent to the PPTMaker API to generate the presentation.

More details about each component will be added soon.
